Connection Instructions:

Share the generated link with the other player.
By allowing clipboard access, the link will automatically get copied.
If clipboard is blocked, manually copy the link to share with other player.
Ask the other player to share their generated answer with you.
Paste their answer in the "Paste Answer" field and press "Start Game".
Note: These steps can be done in split browsers for testing purposes.

Game Instructions:

Arrow Keys - Move the camera around
Mouse Drag - Select units in the selection box
Mouse Left Click - Move units or attack opponent units/structures (click on opponent)
Mouse Right Click - Unselect units
"A" key - Select all your units
"S" key - Scatter selected units
"H" key - Return camera to home structure
"O" key - Move camera to opponent structure
Game Objective - Destroy opponent's structure
Your structure will constantly generate new units
