Instructions:

If you are creating the game (initiator), share the link generated with the other player.
By pressing allow to clipboard, the link will automatically get copied.
If you are joining a game (recieved a link), copy the generated code and share with the initiator.
After the initiatior recives the code paste it in the "paste answer" field and press "Process Answer"

Game Instructions:

Arrow Keys - Move the camera around
Mouse Drag - Select your units
Mouse Click - Move units or attack opponent units/structures
A key - Select all units
S key - Scatter selected units
H key - Return camera to home structure
Game Objective - Destroy opponent's structure
Your structure will constantly generate new units
